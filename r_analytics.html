Factors

Factors can be used as enumerations, and using the terminology of the script, categorical vectors can be broken down into nominal and ordinal sub-categories. Nominal variables are unordered lists: think animal names. Ordinal variables are ordered lists: think sequences such as high to low or bottom to top

The levels() function allows you to assign an alias to a factor e.g. build a vector: my_vector <- c("M","F"); generate a factor from it with my_factor <- factor(my_vector); alias the category with levels(my_factor) <- c("Female","Male")
